src.Atomic.BasicP.Dvlocity_to_Dshift¶

src.Atomic.BasicP.Dvlocity_to_Dshift(p0, v)[source]¶

given Doppler velocity and the line central wavelength/frequency, compute Doppler shift in wavelength/frequency.

Parameters

  • p0 (np.double or array-like) – Frequency in any frequency unit or Wavelength in any length unit

  • v (np.double or array-like) – line of sight velocity, [\(cm/s\)]

Returns

dp – Frequency or Wavelength, same unit with input p0

Return type

np.double or array-like

Notes

in wavelength,

\[d\lambda = \lambda_0 \frac{v}{c}\]

in frequency,

\[d\nu = \nu_0 \frac{v}{c}\]
