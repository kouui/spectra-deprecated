src.Atomic.BasicP.get_Doppler_width¶

src.Atomic.BasicP.get_Doppler_width(p0, Te, Vt, am)[source]¶

Given central wavelength/frequency, relative atomic mass of a line, and the temperature, turbulent velocity, compute the corresponding Doppler Width.

Parameters

  • p0 (np.double or array-like) – Frequency in any frequency unit or Wavelength in any length unit

  • Te (np.double or array-like) – Temperature, [\(K\)]

  • Vt (np.double or array-like) – Turbulent velocity, [\(cm/s\)]

  • am (np.double or array-like) – atomic mass relative to hydrogen atom. [-]

Returns

dp – Frequency/Wavelength Doppler width, same unit with input argument p0

Return type

np.double or array-like

Notes

According to 1.

\[\eta_{0} = (2kT/m + Vt^2)^{1/2}\]

in wavelength,

\[d\lambda = \lambda_0 \frac{\eta_{0}}{c}\]

in frequency,

\[d\nu = \nu_0 \frac{\eta_{0}}{c}\]

References

1

Robert J. Rutten, “Radiative Transfer in Stellar Atmosphere”, pp. 82, 2003.
