src.Atomic.LTELib.get_LTE_ratio¶

src.Atomic.LTELib.get_LTE_ratio(_erg, _g, _stage, _Te, _Ne)[source]¶

Compute LTE population ratio relative to 1st level.

Parameters

  • _erg (numpy.1darray of np.double) – level energy relative to 1st level, [\(erg\)]

  • _weight (numpy.1darray of np.uint8) – statistical weight, [-]

  • _stage (numpy.1darray of np.uint8) – ionization stage, [-]

  • _Te (np.double) – electron temperature, [\(K\)]

  • _Ne (np.double) – electron density, [\(cm^{-3}\)]

Returns

_nRatio – population ratio. [-]

Return type

numpy.1darray of np.double

Notes

We set the population of the 1st level to 1.0, and then compute the population ratio level by level. Saha_distribution is applied whenever there is a jump of ionization stage. Finnaly, we normalize the _nRatio array with respec to the total population.
