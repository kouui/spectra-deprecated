src.Atomic.Collision.get_CE_rate_coe¶

src.Atomic.Collision.get_CE_rate_coe(_CE_fac, _Te, _gi, _dEij)[source]¶

compute the CE rate coefficient.

Parameters

  • _CE_fac (np.double, np.array, (nLine,); scalar) – the coefficient we interpolate from data

  • _Te (scalar) – termperature, [\(K\)]

  • _gi (np.uint8, np.array, (nLine,); scalar) – statistical weight of lower level of the transition, [-]

  • _dEij (np.double, np.array, (nLine); scalar) – excitation energy, [\(erg\)]

Returns

_CEij – collisional excitation rate coefficient, [\(cm^{3} s^{-1}\)]

Return type

np.double, np.array, (nLine); scalar

Notes

For “ECS” (Effective Collisional Strength) 1.

\[n_{e} C_{ij} = n_{e} \frac{8.63e-6 \times (\Omega_{ij} f1 / f2) }{g_{i} T_{e}^{1/2}} \exp{\frac{-dE_{ji}}{kT_{e}} } \quad [s^{-1}]\]

References

1

John T. Mariska, “The Solar Transition Region”, Cambridge University Press, pp. 22, 1992
