src.Atomic.Opacity.Thomson_scattering¶

src.Atomic.Opacity.Thomson_scattering(n_e)[source]¶

Thomson scattering of free electrons (non-relativistic). Returns absorption coefficient instead of absorption cross section.

Parameters

n_e (np.double or array-like) – electron density, [\(cm^{-3}\)]

Returns

kappa – absorption coefficient, [\(cm^{-1}\)]

Return type

np.double or array-like

Notes

from 1.

\[\kappa = \sigma_{T} n_{e}\]

where \(\sigma_{T}\) is the absorption cross section for Thomson scattering .. math:: sigma_{T} = frac{8 pi e^{4}}{3 m_{e}^{2} c^{4}} = 6.6524 times 10^{-25} quad cm^{2}

References

1

Ivan Hubeny, Dimitri Mihalas, “Theory of Stellar Atmosphere: An Introduction to Astrophysical Non-equilibrium Quantitative Spectroscopic Analysis”, Princeton University Press, pp. 149, 2015.
